# Implementa-o-de-Backup-Distribu-do-com-pgBackRest-e-MinIO-em-Ambientes-Virtualizados-com-PostgreSQL
implementação e validação de uma arquitetura de backup distribuído para PostgreSQL em ambiente virtualizado, usando pgBackRest + MinIO (S3) e monitoramento (Zabbix), com testes de backup full/incremental e restore em VM separada.
